# Python Collections Comparison

| Data Structure | Syntax Example | Ordered | Mutable | Allows Duplicates | Access Method |
|---------------|----------------|---------|---------|-------------------|---------------|
| List | `fruits = ["apple", "banana", "orange"]` | Yes | Yes | Yes | Index (`fruits[0]`) |
| Dictionary | `student = {"name": "Shea", "age": 36}` | Yes | Yes | Keys must be unique (values may repeat) | Key (`student["name"]`) |
| Set | `numbers = {1, 2, 3, 4}` | No | Yes | No | Membership (`in`) or loop |
| Tuple | `colors = ("red", "blue", "green")` | Yes | No | Yes | Index (`colors[0]`) |
